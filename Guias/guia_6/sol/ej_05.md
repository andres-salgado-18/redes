#### Ejercicio 5

Si DNS fuera
- centralizado: se generaría demasiada congestión y carga sobre el servidor, lo que afectaría no solo a la obtención de una IP dada una URL, sino que por esto mismo, HTTP se vería afectado
- sin cache: se generaría mayor latencia y carga de red, luego se varía también afectadas las requests de HTTP
- orientado a conexión: generaría el mismo efecto en HTTP, ya que se agregaría un overhead de establecimiento/cierre por cada consulta DNS
Por consiguiente, SMTP también se vería afectado si DNS tuviera alguna de las anteriores características mencionadas
