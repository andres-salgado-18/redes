#### Ejercicio 4

Se van a necesitar las siguientes conexiones TCP
1. Conexión de HTTP del host y la página `ww.smn.gov.ar`
2. Conexión de User agent por medio de SMTP a mail server
3. Conexión entre mail server de salida y destino (aunque no sé si cuenta como iniciada por el usuario)
Serán 2 (3, si se cuenta este último caso).

DNS usa UDP, por lo que las computadoras que se comunican usando UDP serán las involucradas en las consultas DNS
- Cliente
- Servidor DNS (resolver ISP)
- Servidores root DNS, como los de gov, dc, uba, ar, smn

1. Consulta a su resolver local por la dirección de `dc.uba.ar`.
2. El resolver pregunta a los root servers, que indican los servidores del dominio `.ar`.
3. Luego consulta a los servidores de `.ar`, que indican los de `uba.ar`.
4. Finalmente, los servidores autoritativos de `uba.ar` responden con la dirección de `dc.uba.ar`.
