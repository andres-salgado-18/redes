#### Ejercicio 7
Como ya está cacheada la IP del NS de `uba.ar` no se consulta a root, directamente se puede consultar al NS, se le pide su MX
- Se encontrará con la entrada `uba.ar MX`, por lo que habrá una indirección a `mailserver`
- Existe el registro `mailserver` el cual devolverá la IP `208.25.19.2`
Para `milagros.dc.uba.ar` se consulta también a NS de `uba.ar`
- Se encontrará el NS `sevidores.dc.uba.ar` con el valor IP `208.190.1.4`
- Luego se hará un request de `milagros.dc.uba.ar` a `servidores.dc.uba.ar`
- Lo cual encontrará el registro y devuelve la IP `209.190.1.15`
