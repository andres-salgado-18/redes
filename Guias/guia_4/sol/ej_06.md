#### Ejercicio 6

**a)**
Usando [netstat](https://manned.org/netstat) obtengo la información de las conexiones activas.

**b)**
Hay una nueva conexión (o específicamente puerto escuchando en localhost 5555)
```
tcp | 0 | 0 | 0.0.0.0:5555 | 0.0.0.0:* | ESCUCHAR
```

**c)**
Se establece una conexión TCP entre localhost y un puerto genérico con localhost con puerto 5555, por ello se agregan las entradas
```
tcp | 0 | 0 | 127.0.0.1:40312 | 127.0.0.1:5555  | ESTABLECIDO
tcp | 0 | 0 | 127.0.0.1:5555  | 127.0.0.1:40312 | ESTABLECIDO
```


**d)**
Al cerrar el lado del cliente pasa a `TIME_WAIT`
```
tcp | 0 | 0 127.0.0.1:40312 | 127.0.0.1:5555 | TIME_WAIT
```
Esperará una cantidad determinada de segundos y cerrará la conexión
