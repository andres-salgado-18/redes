#### Ejercicio 8


Usando un mecanismo basado en firma digital, se realiza lo siguiente
1. tener par clave privada y pública para todo empleado
2. acordar para todos una función de hash
3. cada mensaje a enviar estará acompañado de una firma, la cual es el resultado de aplicar la función de hash y encriptado con la clave privada del emisor
4. el receptor aplicará la función de hash al mensaje y comparará ese valor con el resultado de desencriptar la firma usando la clave pública del emisor, si coincide, entonces se verifica la autenticación
Nota: Se pueden usar certificados digitales para verificar la propiedad de la clave pública, además de encriptar el mensaje con la clave pública del receptor para que luego el receptor desencripte el contenido con su clave privada
