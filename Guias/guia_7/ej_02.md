#### Ejercicio 2
**a)**


- El documento se le aplica una función $H$ de hash
- Luego tal digest es encriptado usando $K_{A}^-$, generando la firma, luego se tienen tanto firma como documento
- Al final A toma ambos y los encripta usando $K_{B}^+$ para finalmente enviarlo.


**b)**
B debe desencriptar usando su clave privada obteniendo $M + F$, con M es documento original enviado por A y F la firma digital. Como B es el único que puede desencriptar el mensaje C con su clave privada se garantiza confidencialidad. Para verificar la firma, B debe desencriptar la firma usando $K_{A}^+$ y compararla usando mismo hash que el que se aplico al documento, si son iguales queda garantizada la autenticación de A sobre el mensaje enviado.
